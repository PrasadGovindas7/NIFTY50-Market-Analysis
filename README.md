# NIFTY 50 Market Analysis (2020–2025)

This project analyzes the performance, risk, and liquidity behavior of the NIFTY 50 index from 2020 to 2025 using Python and Power BI.

## Objectives
- Analyze long-term market trends and growth
- Compare year-wise performance and extreme return days
- Study market risk using volatility and rolling volatility
- Examine liquidity and market participation through turnover analysis

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- Power BI (interactive dashboards & storytelling)
- Jupyter Notebook

## Project Structure
- `data/` → Raw NIFTY 50 dataset
- `notebooks/` → Python analysis notebook
- `dashboard/` → Power BI dashboard (PDF export)

## Key Insights
- The NIFTY 50 shows a strong long-term upward trend despite short-term volatility.
- Market volatility peaked in 2020 and gradually normalized in subsequent years.
- Year-wise performance highlights recovery and consolidation phases.
- Higher turnover during volatile periods indicates strong market participation.

## How to Run the Analysis
1. Clone the repository
2. Install required libraries
3. Run the Jupyter notebook in the `notebooks/` folder
